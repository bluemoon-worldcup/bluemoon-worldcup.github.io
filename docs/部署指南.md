# 部署指南

项目分两个版本，可独立部署：

- **H5 网页版**（`h5/` 目录）—— 今天就能上线分享，零审核
- **微信小程序版**（`miniprogram/` 目录）—— 需注册+审核，体验更原生

---

## 一、H5 网页版上线（10分钟）

`h5/` 只有两个文件：`index.html` + `data.js`，任何静态托管都能放。

### 推荐方案（按国内访问稳定性排序）

| 方案 | 费用 | 国内访问 | 说明 |
|------|------|----------|------|
| 腾讯云 COS 静态网站 | 几乎为0（流量计费，分享给好友量级约几分钱） | 稳定 | 开通 COS → 建桶 → 开启"静态网站" → 上传两个文件 |
| 阿里云 OSS 静态网站 | 同上 | 稳定 | 同上流程 |
| Netlify / Vercel | 免费 | 时好时坏 | 最省事，但部分地区微信内打开可能慢或失败 |
| GitHub Pages | 免费 | 较差 | 不推荐微信内分享使用 |

> 注意：使用云厂商**默认域名**（如 `xxx.cos-website.ap-shanghai.myqcloud.com`）无需 ICP 备案；绑定自己的域名才需要备案。

### 更新数据

运行 `node tools/update-data.mjs` 重新生成 `h5/data.js`，再把它重新上传覆盖即可。
也可以把 `data.json` 单独托管并在 `index.html` 的 `CONFIG.remoteDataUrl` 里填上地址，页面会自动在线刷新，无需重新部署页面本身。

## 二、微信小程序注册（你还没有 AppID，从这里开始）

1. 打开 https://mp.weixin.qq.com → 右上角「立即注册」→ 选择「小程序」
2. 填一个**没注册过公众号/小程序的邮箱**，激活邮件
3. 主体类型选「**个人**」，填身份证姓名+号码，用本人微信扫码验证
4. 注册完成后：「设置 → 基本设置」里能看到 **AppID**，复制备用
5. 信息补充：
   - 名称：如「蓝月亮看世界杯」（重名的话加后缀）
   - 类目：选「**体育 → 其他体育赛事**」（个人主体可选，无需特殊资质）

## 三、开通云开发（数据自动更新的关键）

> 为什么需要：小程序只能请求有 ICP 备案的域名，国外足球数据 API 都不满足；
> 云函数没有这个限制，可以代替小程序去拉数据。

1. 微信开发者工具（https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html）
   导入本项目 `miniprogram/` 目录，AppID 填你自己的
2. 工具栏点「云开发」→ 开通（按量付费套餐，约 19.9 元/月，世界杯结束可关闭）
3. 记下**环境 ID**，填到 `miniprogram/miniprogram/app.js` 的 `env: 'YOUR_CLOUD_ENV_ID'`
4. 云开发控制台 → 数据库 → 新建集合 `wcdata`，权限设为「所有用户可读，仅创建者可写」
5. 在开发者工具里右键 `cloudfunctions/updateData` → 「上传并部署：云端安装依赖」
6. 云开发控制台 → 云函数 → updateData → 触发器会按 `config.json` 自动创建（每30分钟拉一次数据）
7. 手动触发一次 updateData（云函数控制台「云端测试」），确认 `wcdata` 集合里出现 `latest` 文档
8. 回到模拟器刷新，看到真实数据即成功

## 四、提交审核与发布

1. 开发者工具「上传」→ mp.weixin.qq.com「版本管理」→ 提交审核
2. 审核信息如实填写：世界杯赛程信息展示工具，无用户登录、无支付、无用户生成内容
3. 通常几小时到3天通过 → 「发布」
4. 之后微信里搜名字或扫码即可使用，可直接转发给好友/群

### 审核避坑

- 不要出现"懂球帝/FotMob"等第三方品牌字样
- 页面里已带"数据仅供球迷参考"声明，保留它
- 若被以「体育资讯需资质」驳回：回复申诉说明是赛程日历工具、非新闻资讯，一般可过；
  仍不行可把类目换成「工具 → 信息查询」再提审
