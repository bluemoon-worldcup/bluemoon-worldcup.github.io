# 蓝月亮看世界杯 ⚽

追踪 2026 世界杯上所有曼城球员（含今夏离队的B席、斯通斯，以及德布劳内）的赛程小应用。
FotMob 风格深色界面 + 曼城蓝主题，全部时间显示为北京时间。

**本届世界杯曼城是参赛球员最多的俱乐部：19名现役球员分属12支国家队**，加上德布劳内共20人。
L组堪称"曼城小组"——英格兰(4人)、克罗地亚(2人)、加纳(1人)同组。

## 数据如何更新（三层保障）

1. **H5 浏览器直连**：页面打开/点刷新时直接请求 ESPN 公开接口更新比分和淘汰赛对阵，
   并从 FotMob 公开接口抓取已结束比赛的球员评分（0-10分，FotMob 同款）；
   不可达时自动降级到内置数据
2. **本地脚本**：`node tools/update-data.mjs` 重新生成全部数据文件（含评分），git push 即上线
   （赛程骨架 fixturedownload.com + ESPN 增强 + FotMob 评分，任一失败不影响其余）
3. **小程序云函数**：定时触发器每30分钟自动拉取并写入云数据库（小程序路线已暂停，代码保留）

## 球员评分

- 来源：FotMob 公开接口（`/api/data/matchDetails`），比赛结束后自动出现
- 展示：比赛卡片上每名球员的评分徽章（≥8.0 绿 / 6.5-7.9 黄 / <6.5 橙红），球员页显示世界杯场均分
- 自动抓取失效时可在 `tools/ratings-override.json` 手动维护，格式见文件内说明
- 测试：`node tools/test-ratings.mjs`（用近期已结束的真实比赛验证全链路）

## 数据口径

- 赛程：fixturedownload.com（FIFA 官方104场编号）+ ESPN 非官方API（实时状态）
- 名单：2026-06-11 核实于 Wikipedia 各队26人名单 / 曼城官网 / 各足协官宣
- 福登、萨维尼奥落选名单，多纳鲁马因意大利未晋级无缘，已列入"未参赛球员"折叠区
